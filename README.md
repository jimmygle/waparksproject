## Local Environment Setup

1. Clone the repository.
1. `composer install`
1. `yarn install`
1. `php /vendor/bin/homestead make`
1. `vagrant up`
1. Update `/etc/hosts`:
    ```
    192.168.10.10 waparksproject.local
    ```
1. Visit the local site: http://waparksproject.local/
